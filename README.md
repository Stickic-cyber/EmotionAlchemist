# Emotion Alchemist 🧪✨

Welcome to **Emotion Alchemist**! This is a cozy, interactive sandbox game where you play as an alchemist discovering elemental recipes to soothe and transform wandering negative emotions into joy.

🎮 **[Play the Game Here!](https://stickic-cyber.github.io/EmotionAlchemist/)** 🎮

## How to Play

* **Spawn Elements:** Use the left control panel to summon basic elements like Water, Fire, and Grass.
* **Discover Recipes:** Drag and drop different elements onto each other to combine them. What happens if you mix Water and Fire? Keep experimenting to unlock all items!
* **Heal Emotions:** Different items can calm different negative emotions (Sadness, Anger, Fear, etc.). Move your crafted items near the emotions to heal them.
* **The Goal:** Turn all the little wandering emotions into "Joy" to reach a state of peace.

## Features

* Intuitive drag-and-drop mechanics.
* 11 unique props to craft and discover.
* Smooth, physics-based emotion behaviors and interactions.
* Endless Sandbox mode for continuous relaxation.
